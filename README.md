# Weather Application
Mobile application showing 14-day weather forecast
## Used Technologies
* Retrofit 2 with RxJava 3
* Navigation and Fragment
* Android Widgets
* Location Services
* View Binding
* Recyclerview
* MVVM
* Weather Api = https://www.visualcrossing.com/weather-api
* Google Places Api = Place Autocomplete, Place Details
